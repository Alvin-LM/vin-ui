### VinUI 语言包

目前支持的语言:

| 语言     | 文件名 | 版本     |
| -------- | ------ | -------- |
| 英语     | en-US  | `v0.1.0` |
| 简体中文 | zh-CN  | `v0.1.0` |
| 繁體中文 | zh-TW  | `v0.1.0` |

## 常见问题

### 找不到所需的语言包？

如果上方列表中没有你需要的语言，欢迎给我们提 Pull Request 来增加新的语言包。

### 业务代码如何实现国际化？

可以使用 [vue-i18n](https://github.com/kazupon/vue-i18n) 来实现。
